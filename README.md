README : Food Delivery Website
Project Overview

This project is a Food Delivery Web Application where users can explore food items, add them to a cart, and place orders easily.

->Features
. Add to Cart functionality
. Browse food items
. Search option
. Order placement
. Responsive design
. Technologies Used
   ->HTML
   ->CSS
   ->JavaScript
